Calculator (Python)

A smart command-line scientific calculator built using Python.  
It supports arithmetic operations, trigonometric functions, logarithms

~Features

~Basic arithmetic operations: 
  `+  -  *  /`
~Supports multiple input styles:
  `5x3`, `5X3`, `5×3`
~Trigonometric functions:
  `sin`, `cos`, `tan`
  `sec`, `cosec`, `cot`
~Smart input parsing:
  `sin30` - automatically converted to radians
  `3(2+3)` - handled as `3*(2+3)`
~Mathematical functions:
  `log10`
  `sqrt16`
~Previous answer feature:
  Use `Ans` in next calculation
~History tracking:
  `history` - shows previous calculations
  `clear` - clears history
  `exit` - exit from the calculator
~Error handling for invalid inputs
